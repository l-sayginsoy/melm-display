Hintergruende (optional)

Du kannst eigene Hintergrundbilder ablegen. Der Code sucht nach Pfadmustern wie:

bg/winter/clear/01.jpg
bg/summer/rain/02.jpg

Wenn keine passenden Bilder vorhanden sind, nutzt die Seite automatisch einen Wetter Farbverlauf.

Tipp
Lege zuerst nur wenige Bilder ab, pruefe am Monitor, und erweitere dann.
